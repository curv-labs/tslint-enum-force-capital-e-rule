# tslint-enum-force-capital-e-rule

It is a TSLint rule to enforce a capital e at the start of enum declarations.

## Installation

```bash
npm i -D @curv-labs/tslint-enum-force-capital-e
```

In `tslint.json` config

```json
{
  "extends": [
    "tslint-enum-force-capital-e-rule"
  ],
  "rules": {
    "enum-force-capital-e": true
  }
}
```
